## Installation

Clone this repository and then use the package manager npm to install dependencies.


```bash
npm install
```

## Setup .env example

Create .env file in your root project folder.

```env

PORT=8081
DB_HOST=localhost
DB_USER =root
DB_PASSWORD=your_password
DB_NAME=hiring_channel
SECRET_KEY= 'iniprivatekey'
AUTH_EXPIRES=1h

```

## Run the app

Development mode

```bash
npm run dev
```

Deploy mode

```bash
npm start
```
