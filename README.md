| Endpoint      | Method           | Info |
| ------------- |:-------------:|:---|
| /v1/users  | `POST` `GET` `PUT` `DELETE` | User |

* Express JS
* Socket.io
* **JWT** for Authentication
* **Nodemailer** for Mailer
* **Multer** for Upload file
